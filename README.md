## Portfolio

Responsive portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

### Run locally

```bash
npm install
npm run dev
```

### Edit content

- Profile, socials, skills, and projects live in `src/data.js`.
- Replace placeholder images and links with your own.

### Tech stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Icons
